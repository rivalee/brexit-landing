Landing page refresh

Designs for user research

  1. Tabs: landing page with tabs

  2. Links: landing page with content split into pages

High-level hypotheses informing these designs

  • There's evidence that users are seeking more specific, specialised guidance than the Brexit checker is providing them with. The Brexit checker has been moved below the guidance section. It's still there if users wish to use it, but the visual priority is on the guidance rather than the checker.
  • There are two main user groups: businesses and citizens. It's helpful if we split the content for these two user groups so it's easier for users to find content specific to their needs.

Tabs vs pages

  • Content: tabs (design 1) are meant for relatively short and lightweight content: if we end up adding more specialist content, splitting content into pages (design 2) will give us more flexibility
  • Usability: we're not sure if users will find tabs or pages easier to use
